Contributing to Lobos

As this project hasn't yet reached the magical 1.0 version, this is the best time for anyone to contribute their idea and/or code. If you want to participate, the first step would be to visit the online repository on github.

You can also visit the Google Group to ask questions about the code, start discussion about features, bug fixing or any subject related to the project.

Issue Tracker

All issues, suggestions and requests should be added to the project's issue tracker.

Wiki

Finally a wiki is also available. Its main purpose is to discuss upcoming features in more depth before being implemented.
